ID e CLASS

CLASS

Existe duas caracteristicas principais do class:

  • È possivel ultilizar o mesmo class em vários elementos diferentes
  • É possivel ultilizar várias classes no MESMO elemento

ID

existe duas caracteristicas principais do ID que são contrastante com as caracteristicas do class:

  • não é possivel utilizar o mesmo ID em vários elementos diferentes.
  • não é possivel utilizar VÁRIOS IDs no mesmo elemento

o ID funciona como um CPF. não podem existir dois iguias e a mesma pessoa não pode ter vários CPFs
